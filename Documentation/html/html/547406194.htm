EntropyFileRNG Class

It is recommended, at least one time, to prompt user for random keyboard input, mouse input, perhaps poll some internet random sources, and every possible available entropy source, and use it to seed the EntropyFileRNG. The EntropyFileRNG will save this to disk, and upon every time an application is launched that uses EntropyFileRNG, the file will always be reseeded by itself, and will serve PRNG bits
